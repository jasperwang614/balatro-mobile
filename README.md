# Balatro Mobile — 小丑牌 HTML5 复刻版

> 单文件、零依赖、移动端适配的 Balatro 复刻。
> 浏览器打开即玩，支持存档。

## 快速开始

用浏览器打开 index.html 即可。

或在手机上：
1. 把 index.html 传到手机
2. 用浏览器打开
3. 添加到主屏幕 → 全屏体验

## 游戏系统

### 牌型（16 种）

| 牌型 | 底分 | 牌型 | 底分 |
|---|---|---|---|
| 高牌 | 50×10 | 同花 | 350×40 |
| 对子 | 100×20 | 葫芦 | 400×40 |
| 两对 | 200×20 | 四条 | 600×70 |
| 三条 | 300×30 | 同花顺 | 1000×80 |
| 顺子 | 300×40 | 五条/同花葫芦/同花五条 | 1200+ |
| 全奇数 | 400×40 | 全质数 | 400×40 |
| 全偶数 | 400×40 | 等差 | 200×40 |

### 小丑牌（46 张）

- **倍率型**：小丑牌 +40、贪婪 +30/♣...
- **乘算型**：Cavendish ×30、黑板 ×6...
- **条件型**：牌型含对子 ×20、出牌 ≤3 张 +120...
- **重触发**：重锤（全小丑+3次）、回声（强化牌+3次）
- **累积型**：搭巴士（连续无对子+1）、备用裤（每次两对+2）、星座（用星球牌+1）

### Boss（8 个）

獠牙（手牌-1）、高墙（目标×1.5）、钩子（弃牌归零）、梅花（只有♣得分）、游鱼、灵能、尖刺、血口

### 商店

| 类型 | 数量/店 | 说明 |
|---|---|---|
| 小丑牌 | 3+ 张 | 数据驱动，可重复购买叠加 |
| 星球牌 | 2 张 | 升级牌型底分 |
| 塔罗牌 | 2 张 | 强化/复制/删除/变牌 |
| 优惠券 | 1 张 | 永久增益（手数/弃牌/折扣） |

### 难度

| 难度 | 倍数 | 手数 | 弃牌 | 初始$ |
|---|---|---|---|---|
| 简单 | ×0.8 | +1 | +1 | 30 |
| 普通 | ×1.0 | 4 | 3 | 20 |
| 困难 | ×1.3 | -1 | -1 | 10 |
| 极限 | ×1.6 | -2 | -2 | 5 |

### 卡牌强化

- **加成牌**（bonus）：+300 筹码
- **倍率牌**（mult）：+40 倍率
- **玻璃牌**（glass）：×8 倍率
- **钢铁牌**（steel）：手持 ×3
- **石头牌**（stone）：+500 筹码
- **万能牌**（wild）：任意花色

### 版本 Edition

- 镀膜 ×1.5 | 全息 ×2.0 | 彩印 ×3.0

## 操作

| 操作 | 方式 |
|---|---|
| 选牌 | 点击切换选中 |
| 出牌 | 拖拽到出牌区 / 点击出牌按钮 |
| 弃牌 | 选中后点弃牌 |
| 排序 | 按点数/花色 |
| 商店 | 点击购买 |

## 技术

- 纯 HTML/CSS/JS，零依赖
- CSS 变量主题系统
- 移动端拖拽（pointer events）
- localStorage 存档
- 数据驱动：所有牌型/小丑/Boss/塔罗 均为 JSON 配置
- 无限延伸：超出预设底注后指数增长
